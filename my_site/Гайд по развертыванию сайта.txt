Развертывание сайта с панелью Спанель
1. в Спанели заходим в Программное обеспечник - Настройки Python приложений
2. Создать приложение
3. Версия Python выбираем максимальную
4. Корневой каталог приложения - обязательно должен начинаться public_html/'любое_имя'
5. Создать
6. Копируем путь к вертуальному окружению
7. в Спанели - Расширенный - Терминал (если его нет то пишим в тех поддержку чтобы открыли)
8. в Терминале - вставляем путь см.6
9. pip install --upgrade pip
10. pip install django 
11. django-admin startproject project . (вместо project вставляем имя своего проекта, которое в PyCharm)
12. откройте файл passenger_wsgi.py в менеджере файлов и удалите всё содержимое, после пропишите:
	from project.wsgi import application  (укажите название своего проекта вместо project)
13. теперь в директории установки создайте папку(можете укажите определённый путь для директории): assets
14. далее перейдите в папку установки и создайте две папки: static и media
15. Копируем с компьютера через FTP сервер (данные подключения смотри в письме от хостинга) папки
	с данными проекта, приложения, manage.py, requaements.txt. Все заменяем
16. В терминале pip install -r requirements.txt Это установит все зависимости
17. в конфигурационном файле settings.py в троке ALLOWED_HOSTS  пропишите свой домен 
	DEBUG = False
	ALLOWED_HOSTS = ['vivog.pp.ua', 'www.vivog.pp.ua']
	
	STATIC_URL = '/static/'
	MEDIA_URL = '/media/'
	STATICFILES_DIRS = [BASE_DIR / "assets"]
	MEDIA_ROOT = '/home/h52977c/public_html/resume/media' 
	STATIC_ROOT ='/home/h52977c/public_html/resume/static'
	
	Пути меняем если что!!!
	СОХРАНЯЕМ
18. в терминале python manage.py collectstatic и перезагружаем приложение в Спанели. Сайт уже должен работать
	если нет то пишим в тех поддержку
19. Подключаем базу данных из гайда
20. Если что то не пашет то пишим в тех поддержку
